Aula de GIT

Lista de comnados

  • git status: mostra status do projeto no git.
  • git add [file]: adiciona o arquivo (ou mudanças) no controle.
  • git rm -- cached [file]: remove o arquivo (ou alteração) no arquivo.
  • git commit: cria um novo commit na linha do tempo do projeto com as alterações da Staging Area<./li>
  • git log: mostra todos os commmits feitos.
  • git pull: atualiza os dados locais com os commits do repositório
  • git push: atualiza os dados remotos com os commits do local
